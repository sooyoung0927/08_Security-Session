접근 권한 안내

ADMIN 권한으로도 접근 가능

빠른 작업

사용자 메뉴

사용자 권한으로 이용 가능한 주요 메뉴와 테스트용 이동 링크입니다.

홈 화면

메인 페이지로 돌아가 로그인 상태와 권한별 메뉴를 다시 확인할 수 있습니다.

로그인 페이지

로그인 화면으로 이동하여 인증 흐름을 다시 테스트할 수 있습니다.

회원가입

새로운 계정을 생성하고 세션 기반 인증 과정을 확인할 수 있습니다.
